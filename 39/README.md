## 随机森林算法 `demo`
随机森林实际上是一个包含多个决策树的分类器，每一个子分类器都是一棵CART分类回归树。所以随机森林既可以做分类，也可以做回归。当它做分类的时候，输出结果是每个子分类器的分类结果中最多的那个。当它做回归的时候，输出结果是每棵CART树的回归结果的平均值。
